vamos começar pelo básico

Commandos básicos:

W - andar para frente

S - andar para trás

A - andar para a esquerda

D - andar para a direita

E - abrir inventário

Espace (Espaço) - pular

Segurar CTRL (muda da forma que você configurou) - correr (junto com w)

ESC - pausar jogo

Primeiros Recursos

para começar vamos pegar um pouco de madeira e tranformar em tábuas para que nós podemos criar nossas ferramentas iniciais
